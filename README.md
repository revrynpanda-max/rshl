## 🚀 Quick Start

Ensure you have the Rust toolchain and Node.js (v18+) installed.

### 1. Start the Oracle Engine (Rust)
```powershell
cargo run --release --bin kai
```

### 2. Start the OpenJarvis Backbone (Python)
```powershell
uv run jarvis serve --port 8080
```

### 3. Start the Discord Gateway (Node.js)
```powershell
cd tools/oracle-discord
node index.mjs
```
